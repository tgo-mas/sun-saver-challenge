# Arena Solar

### O seu incentivo para a economia e geração sustentável de energia!

O Arena Solar é um jogo que mistura ciência de dados com geração de energia solar residencial, fornecendo metas e desafios para otimizar sua economia de energia, aumentando a produção e aliviando a conta de luz, enquanto nos ajuda a trilhar um caminho sustentável.

A API é feita com Django e Pandas, enquanto o front-end é uma aplicação React com Vite e TailwindCSS. Esse projeto foi construído para a Hackaton do Bootcamp Python Aplicado à Ciência de dados e IA para o Setor Elétrico Brasileiro.
